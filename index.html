Draw Module

Page still in progress.

The Draw Module contains everything you need to set the pixels on the screen. This includes shapes, images, particles, and more.

The Draw Module creates multiple canvases called "Layers." At the end of every frame, all of the Layers are combined and drawn to the screen.

The draw Module can be accessed with MDog.Draw.method().

$METHODS

Drawing Methods

Method Description
Draws a circle.
Draws a circle using a distance method.

Parameters:
x, y: The integer center of the drawn circle.
radius: The integer radius of the drawn circle.
color: Hex code for the circle fill.

Settings (optional):
layer - An integer defining the canvas to draw to. If the layer has never been drawn to, a new canvas will be created. Default: 0.
Clears the screen/layer to a color.
Clears the screen (or specific layer) to a color (black by default).

Settings (optional):
color - Hex code for background clear.
layer - An integer defining the canvas to draw to clear. If no layer is given, all layers will be cleared.
Draws an image.
Draws an image.

Parameters:
fileName The name of the file to be drawn. This is auto-prefixed with "assets/".
x, y: The integer top left of the image.

Settings (optional):
layer - The canvas to draw to. If the layer specified layer has never been drawn to, a new canvas will be created. Default: 0.
scale - How many pixels each image pixel maps to. Default: 1.
scaleX - How horizontally stretched the image should be. Default: this.scale.
scaleY - How vertically stretched the image should be. Default: this.scale.
offsetX - A horizontal offset into the image of where to crop. Default: 0.
offsetY - A vertical offset into the image of where to crop. Default: 0.
width - The width of the image relative to this.offsetX. Default: image.width.
height - The height of the image relative to this.offsetY. Default: image.height.
flipX - An integer defining the canvas to draw to. If the layer specified layer has never been drawn to, a new canvas will be created. Default: 0.
flipY - An integer defining the canvas to draw to. If the layer specified layer has never been drawn to, a new canvas will be created. Default: 0.

Draw Module

Page still in progress.

The Draw Module contains everything you need to set the pixels on the screen. This includes shapes, images, particles, and more.

The Draw Module creates multiple canvases called "Layers." At the end of every frame, all of the Layers are combined and drawn to the screen.

Drawing Methods

Method Description
circle(x, y, radius, color, settings) Draws a circle.
clear(settings) Clears the screen/layer to a color.
image(fileName, x, y, settings) Draws an image.
interactable(interactable) Draws an interactable.
line(x1, y1, x2, y2, color, settings) Draws a line
particleSystem(particleSystem) Draws a particle system.
point(x, y, color, settings) Sets a pixel.
polygon(points, color, settings) Draws the outline of a polygon.
polygonFill(points, color, settings) Draws a filled polygon.
rectangle(x, y, width, height, color, settings) Draws the outline of a rectangle.
rectangleFill(x, y, width, height, color, settings) Draws a filled-in rectangle.
setActiveLayer(layer) Sets the default draw layer.
text(x, y, text, color, settings) Draws text.
threeDeeScene(threeDeeScene) Draws a threeDeeScene
translate(x, y, settings) Shifts the screen.
translateX(x, settings) Shifts the screen horizontally.
translateY(y, settings) Shifts the screen vertically.

Setup Functions

Method Description
loadFont(fontName, fontURL) Adds a custom font.
setBackgroundColor(color) Sets the color of the dom body.

Info Methods

Method Description
getScreenHeightInArtPixels() Gets the height of the screen.
getScreenWidthInArtPixels() Gets the width of the screen.
getHalfScreenHeightInArtPixels() Gets the int height of half of the screen.
getHalfScreenWidthInArtPixels() Gets the int width of half of the screen.
measureText(text, settings) Gets the length of theoretical text.
